JQuery笔记

Veröffentlicht am 2018-03-09

使用jQuery前要先在script标签中外部引入jQuery的js文件

1.一些常用方法

(1)$() 里面可以写css选择器、 原生dom、 jquery对象、null/undefined、函数function(){}、selector/content等。

注：$('ul>li:first');    选出来的是jquery对象
    $('ul>li:eq(2)');
    $('ul>li:odd/even');

filter('.box/[title=hello]');
has(); find(); eq(); not();  is();->true/false

(2)html()
可以在括号内写入html类型内容
当取值时，是取第一个赋值赋一组

(3)css()
取值相当于 getComputed , 赋值相当于dom.style.xxx
赋值赋一组 取值取一个(第一个 颜色会在内部转换成rgb)

(4)attr() prop() 取值赋值和上面一样

(5)next(); 下一个兄弟节点
prev(); 上一个兄弟节点
index(); 当前兄弟节点中的索引
siblings() 当前元素节点的所有兄弟节点
prevAll() 当前元素上面的所有兄弟节点
nextAll() 当前元素下面的所有兄弟节点
prevUntil()
nextUntil() ->掐头去尾选中元素,传jquery对象或dom

(6)addClass() removeClass() 参数可以是function

(7)A.insertBefore(B) A插入到B的前面 (有剪切的功能)
A.before(B) 在A的前面插入B (有剪切的功能)

(8)A.appendTo(B): A添加到B的里面 (添加到最后一个子元素后)
A.append(B): A里添加B (同上)

(9)A.prependTo(B): 把A添加到B元素中的最前面
A.prepend(B): 把B添加到A元素中的最前面

(10)remove() 从DOM中删除所有匹配的元素,定的事件，附加的数据等都会被移除。
detach() 从DOM中删除所有匹配的元素,所有绑定的事件、附加的数据等都会保留下来。

(11)on() 注册事件
off() 解除绑定

(12)scrollTop() 滚动出去的高度: $(window).scrollTop()

(13)innerWidth 宽度: content padding
outerWidth 宽度: content padding border
宽度: 参数为true的时候 求出的宽度带有margin

(14)offset().left/top 相对于文档
position().left/top 相对于最近有定位的父级

(15)parent() 找当前元素的第一个父节点，不管匹不匹配都不继续往下找
offsetParent() 找其最近有定位的父级元素
parents() 找其所有的父级元素
closest() 是找当前元素的所有父节点 ，直到找到第一个匹配的父节点

(16)val() 返回或设置被选元素的值。
元素的值是通过 value 属性设置的。该方法大多用于 input 元素

(17)each() 遍历 obj.each(function (index, demo) {})

(18)end() 回退到链条中的上一个操作步骤

(19)clone() 克隆节点 参数为true的时候 事件也能克隆

(20)A.wrap(B) 分别使用B来包裹A中的每个元素
A.wrapInner(B) 分别使用B来包裹A中的每个元素中的内容
A.wrapAll(B) 使用B来包裹A中的所有元素
unwrap() 解除包装 结构化标签不能删除

(21)$(selector).animate(styles, speed, easing, callback)

通过CSS样式将元素从一个状态改变为另一个状态。CSS属性值是逐渐改变的，这样就可以创建动画效果。

easing    规定在不同的动画点中设置动画速度的 easing 函数
callback  animate函数执行完之后，要执行的函数

(22)stop() 阻止当前运动继续后续运动
stop(true) 阻止后续所有运动
stop(true, true) 停止当前所有运动 停在当前阶段运动的最终目标点
stop(false, true) 跳过当前运动并到达此阶段最终目标点，执行后续运动
finish() 停止当前运动 并且达到最后运动的最终目标点

(23)slideUP(speed, callback) 通过滑动效果隐藏被选元素
slideDown() 通过滑动效果显示隐藏的被选元素
slideToggle() 通过滑动效果切换元素的可见状态
ToggleClass() 切换元素的class状态

2.关于事件

(1)事件对象 e
e.pageX 获取鼠标当前相对于页面的坐标
e.clientX
e.which 获取在鼠标单击事件中鼠标的左、中、右键
e.button

(2)e.preventDefault(); 取消默认使件
e.stopPropagation(); 阻止冒泡
都可以return false;

(3)one() 绑定事件,且只执行一次

(4)$(“.class”).click(function(){})

(5)$(selector).trigger(event, [param1,param2,…]) 事件触发器
event 规定指定元素要触发的事件,可以是使自定义事件
[param1,param2,…] 可选，传递到事件处理程序的额外参数

3.JQuery工具方法

注：此类方法定义在jQuery函数($)上面

(1)$.type() 判断参数的数据类型

(2)$.trim 消除字符串开始和末尾处空格

(3)$.proxy(function, context) 改变this指针方向
参数1：已有的function, 参数2:函数执行期上下文

(4)$.parseJSON() 字符转换成json
注：要求非常严格，外部是单引号，内部属性是双引号，不需要回车符。

(5)$.makeArray(obj) 把类数组转换成数组

(6)$.Callbacks(“once”) 回调函数列表
.add() 给回调对象添加函数
.fire() 触发添加的函数

可传四个参数
once: 确保这个回调列表只执行（ .fire() ）一次
memory: 保持以前的值，将添加到这个列表的后面的最新的值立即执行调用任何回调
unique: 确保一次只能添加一个回调，列表中没有重复的回调
stopOnFalse: 当一个回调返回false 时中断调用

(7)$.deferred()

//var df = $.Deferred();  //将df变为一个有状态的对象,写在外面容易被人改变状态

function a() {
    var df = $.Deferred();  //将df变为一个有状态的对象
    setInterval(function () {
        df.resolve("ok");  //将df的状态变为成功
        //df.reject("err");  //将df的状态变为失败
        //df.notify();     //将df的状态变为进行中
    }, 1000)
    return df.promise();   //返回一个只读的df，防止外部修改df的状态
}

a().done(function (data) {  //成功的回调函数

}).fail(function (err) {   //失败的回调函数

}).progress(function () {})  //状态为进行中的回调函数

(8)$.when(obj1, obj2, …)
所传的参数都为有状态的对象，只有当所有对象都变成成功状态后才会调用回调函数。

(9)$.ajax({})

所传对象的属性包括：
    1.type: GET POST 访问形式
    2.url：访问路径
    3.data:传输数据
    4.context：回调函数的作用域
    5.aync：是否异步 默认true
    6.cache：是否缓存 默认true
    7.dataType：一般请求不予填写 jsonp请求是需要填写为jsonp
    8.success：访问成功是触发的函数 参数 是返回数据
    9.error：返回失败的时候触发的函数
    10.crossDomain：true跨域 默认false

4.JQuery扩充方法

(1)$.extend

第一种用法 $.extend(obj)  只有一个参数时，相当于给JQuery函数添加工具方法

第二种用法 $.extend( [deep ], target, object1 [, objectN ])
    用于将一个或多个对象的内容合并到目标对象
    deep    可选，Boolean类型 指示是否深度合并对象，默认false。如果为true，且多个对象的某个同名属性也都是对象，则该"属性对象"的属性也将合并。
    target    Object类型 目标对象，其他对象的成员属性将被附加到该对象上。
    object1    可选，Object类型 第一个被合并的对象。
    objectN    可选，Object类型 第N个被合并的对象。

(2)$.fn.extend(obj)
其中$.fn = $.prototype，obj里可以添加多个属性函数。
jQuery的实例可以使用这些添加的方法。
